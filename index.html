Photoshop Battle, 2019
This assignment was given to familiarize students with photoshop tools and prepare them for the Dada project. I learned how to use selection tool and how to mask images.

Dada Project: Summer Dream, 2019
This assignment was given to help students understand what Dada Art. I started to work on the composition from the materials I was given. I became more familiar with the masking tool in Photoshop during this project.

Sequential Art: Free and easy wandering part 1, 2019
This assignment was given to teach us the drawing tools in Illustrator and use them to make vector sequential art. I explored different tools for drawing in illustrator. For example, the pen tool is very useful for drawing lines l and pathfinder tool is handy cutting shapes. I also learned how to arrange my art sequentially..
